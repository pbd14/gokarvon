Keyboard Navigation Test

Try using the keyboard to navigate the language dropdown:

  1. Press Tab to focus on the language dropdown button
  2. Press Enter or Space to open the dropdown
  3. Use Up and Down arrow keys to navigate between languages
  4. Press Enter to select a language
  5. Press Escape to close the dropdown without selecting
